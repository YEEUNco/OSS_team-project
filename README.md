## 🍔 miniproject 주제
> 서울 맛집(음식점)을 찾아주는 프로그램

## 🍔 miniproject 소개
> 서울에 있는 맛집을 널리 알리고, 후기를 남기고, 검색해보세요. 한식, 양식, 중식, 일식 다양한 맛집을 찾아보고 추천받을 수 있습니다. <br> 기본적인 기능인 CRUD와 파일 I/O, 검색기능에 더해 별점 후기 기능까지 넣을 예정입니다.

## 🍔 miniproject 가지고 있는 대략적인 기능 설명
>- 맛집 등록, 수정 기능
>- 종류와 구 단위로 맛집 소개
>- 맛집 별 추천 메뉴 소개
>- 영업 시간, 휴무일 안내 
>- 별점, 리뷰 작성
